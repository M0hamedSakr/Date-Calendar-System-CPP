# Date & Calendar System (C++)

A C++ project that handles date operations such as:
- Date validation
- Day of week calculation
- Month & Year calendar display
- Date comparison
- Vacation days calculation
- Period overlap checking
- Adding days to a date

## Features
- Leap year handling
- Calendar printing (month / year)
- Vacation calculation excluding weekends
- Date difference calculation
- String to date conversion (dd/mm/yyyy)

## Technologies
- C++
- Standard Library (iostream, iomanip, ctime, vector, string)
